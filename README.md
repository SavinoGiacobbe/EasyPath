# EasyPath

**EasyPath** is a QGIS plugin designed to support the analysis and improvement of urban accessibility — both physical and environmental — in Mediterranean cities.  
It provides GIS-based decision support tools for evaluating urban facilities accessibility and simulating intervention scenarios.

---

## Features
- Calculates physical and environmental accessibility indicators
- Generates optimal routes based on customizable criteria
- Allows simulation of urban accessibility improvement scenarios
- Produces cartographic and statistical outputs (maps, charts, reports)

---

## Installation
1. Download the latest version of **EasyPath.zip** from the [QGIS Plugin Repository](https://plugins.qgis.org/).
2. In QGIS, open **Plugins → Manage and Install Plugins → Install from ZIP**.
3. Select the ZIP file and click **Install Plugin**.

---

## Usage
After installation, you will find **EasyPath** under the **Plugins** menu or in the toolbar.  
Click the EasyPath icon to open the main dialog and start your analysis.

---

## Requirements
- QGIS ≥ 3.22
- No additional Python libraries required (uses built-in PyQGIS and PyQt)
